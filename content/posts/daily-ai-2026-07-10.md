## ⭐ 今日最值得关注

### 1. OpenAI 发布 GPT-5.6：更强智能，更高性价比

OpenAI 发布了新一代旗舰模型 GPT-5.6，主打"每个 token 都更聪明"，在性能/美元指标上显著提升，并支持按需扩展能力。同时宣布 GPT-5.6 成为 Microsoft 365 Copilot 的首选模型，为 Word、Excel、PowerPoint、Chat 和 Cowork 提供更强的 AI 能力。这意味着 GPT-5.6 已经进入微软的生产环境，数百万 M365 用户将率先体验到这一升级。

- 原文：[GPT-5.6: Frontier intelligence that scales with your ambition](https://openai.com/index/gpt-5-6)
- M365 集成：[GPT-5.6 is now the preferred model in Microsoft 365 Copilot](https://openai.com/index/gpt-5-6-preferred-model-microsoft-365-copilot)

### 2. ChatGPT Work：OpenAI 打造"最雄心勃勃工作"的 Agent

OpenAI 推出 ChatGPT Work，一个能跨应用和文件执行操作、可在一个项目上持续工作数小时的 Agent。它能把一个目标转化为完成的工作——从规划到执行到交付。这标志着 ChatGPT 从"对话助手"向"工作代理"的关键转变，直接瞄准了企业级复杂任务的自动化市场。

- 原文：[ChatGPT is now a partner for your most ambitious work](https://openai.com/index/chatgpt-for-your-most-ambitious-work)

### 3. Meta 推出 Muse Spark 1.1，以四分之一价格杀入 AI 模型市场

Meta 发布旗舰模型 Muse Spark 1.1，在 Agent 能力、编程和多模态等多项测试中成绩超过 Google 的 Gemini 模型。扎克伯格透露，Meta API 定价约为 OpenAI 和 Anthropic 同类顶级模型的四分之一。这是 Meta 首次向企业收取模型使用费用，正式加入 AI 模型价格战，标志着 API 市场竞争进一步白热化。

- 来源：[华尔街见闻报道](https://wallstreetcn.com/articles/3776599)

## 📰 快讯

### OpenAI 推出 GPT-5.5 生物安全漏洞赏金计划

OpenAI 发布 GPT-5.5 Bio Bug Bounty 计划，聚焦生物安全领域的模型漏洞披露。这是 OpenAI 在安全治理方面的又一举措，旨在通过社区力量发现和修复与生物风险相关的模型安全问题。

- 原文：[GPT-5.5 Bio Bug Bounty](https://openai.com/index/bio-bug-bounty)

### Google 扩展 Gemini API Managed Agents：后台任务、远程 MCP 等

Google 为 Gemini API 的 Managed Agents 增加了后台任务、远程 MCP 支持等新能力，帮助开发者构建可靠的生产级 Agent。这一更新降低了 Agent 应用的运维复杂度，让开发者可以更专注于业务逻辑而非基础设施。

- 原文：[Expanding Managed Agents in Gemini API](https://blog.google/innovation-and-ai/technology/developers-tools/expanding-managed-agents-gemini-api/)

### HuggingFace × NVIDIA：开放 Agent 数据，为什么合成数据是关键

NVIDIA 在 HuggingFace 发文阐述 Agent AI 需要开放数据的理由：真实世界不像基准测试，Agent 必须能从 API 调用失败、未见过的流程中恢复。文章提出从"模型权重"到"训练数据"的范式转变，用合成数据规模化 Agent 的工具调用、多步推理、用户模拟等能力。

- 原文：[Data for Agents](https://huggingface.co/blog/nvidia/open-data-for-agents)

### HuggingFace 推出 vLLM 原生速度的 transformers 建模后端

HuggingFace 宣布 transformers 的 vLLM 后端已经达到甚至超过自定义 vLLM 实现的速度。模型作者可以直接使用 transformers 中的模型实现获得超快推理速度，无需额外迁移成本。升级仅需 `uv pip install --upgrade vllm --torch-backend auto`。

- 原文：[Native-speed vLLM transformers modeling backend](https://huggingface.co/blog/native-speed-vllm-transformers-backend)

### LeRobot v0.6.0：想象、评估、改进——闭合机器人学习环路

HuggingFace 发布 LeRobot v0.6.0，引入"世界模型策略"（VLA-JEPA、FastWAM、LingBot-VA）让策略在行动前"想象"未来，新增奖励模型 API 判断机器人是否成功，部署 CLI 将失败转化为训练数据，以及六个新仿真基准。还加入了深度感知、VLM 驱动的数据集标注和云端训练支持。

- 原文：[LeRobot v0.6.0: Imagine, Evaluate, Improve](https://huggingface.co/blog/lerobot-release-v060)

### 前美联储主席伯南克加入 Anthropic 治理机构

诺贝尔经济学奖得主、前美联储主席本·伯南克正式加入 Anthropic 长期利益信托机构，拥有实质性公司治理权（可任免董事会多数席位）。此举被视为 Anthropic 在 IPO 前强化公共使命可信度的关键举措。

- 来源：[华尔街见闻报道](https://wallstreetcn.com/articles/3776607)

### PrismML：270 亿参数大模型首次在 iPhone 上全量运行

初创公司 PrismML 宣布将阿里巴巴开发的 270 亿参数开源模型 Qwen 3.6 压缩至 iPhone 17 Pro 本地运行，刷新移动端 AI 模型规模纪录。据报道苹果已与其接触，该技术有望成为苹果强化设备端 AI 的关键突破。

- 来源：[华尔街见闻报道](https://wallstreetcn.com/articles/3776594)
